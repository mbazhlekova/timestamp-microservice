Timestamp Microservice

The Timestamp Microservice takes in a date in either a natural format (June 17, 2017) or a Unix timestamp (1497675600) and returns the Unix timestamp and natural date if input is valid and null if input if invalid.

Example input:

http://localhost:3000/1497675600
http://localhost:3000/June%2017,%202017

Example output

{"unix":"1497675600","natural":"June 17, 2017"}
